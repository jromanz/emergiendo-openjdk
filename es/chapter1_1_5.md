#### J2SE 1.4 (6 de febrero de 2002)

	

Con el codigo nombre **Merlin**, fue la primera versión para la plataforma desarrollada por el **JCP** como la JSR 59:


* La palabra clave assert(JSR 41)
* Expresiones regulares
* Encadenamiento de excepción permite que una excepción de mayor nivel encapsule una excepción de menor nivel.
* Soporte al Protocolo de internet versión 6 (IPv6)
* Llamadas de IO (llamado de NIO) nuevos Input/Output (JSR 51)
* API de loggin (JSR 47)
* API para leer y escribir imagenes en formatos como JPED e PNG
* Integración con XML y XSLT (JAXP) en JSR 63
* Nuevas integraciones con extensiones de seguridad y criptografia (JCE, JSSE, JAAS).
* Java Web Start incluído (JSR 56).
* API de preferencias (java.util.prefs)
